# 💰 Financial AI Agent using Phidata & Open Source LLMs

An intelligent **Financial AI Agent** built using **Phidata** and **open-source LLMs via Groq**, capable of answering financial queries, analyzing data, and assisting with decision-making.

---

## 🚀 Features

- 📊 Financial question answering (stocks, trends, etc.)
- 🧠 LLM-powered reasoning using Groq models
- ⚡ Fast responses with streaming support
- 🔧 Tool integration (via Phidata)
- 🏗️ Modular and extensible architecture

---

## 🛠️ Tech Stack

- Python  
- Phidata (phi framework)  
- Groq API  
- Open-source LLMs (LLaMA models)  
- Virtual Environment (venv)
